# Cursor Clone

A static front-end clone of the [Cursor](https://cursor.com) homepage, built with plain HTML and CSS.

## Project Structure

```
assignment-css1-cursor-clone/
├── index.html        # Main HTML file
├── styles.css        # Stylesheet
└── aasets/           # All images and SVG assets
    ├── General Logos/
    ├── Logo Garden/
    ├── Fronteir Section/
    ├── Review-avatars/
    ├── Section-1-image.png
    ├── section-2-image.png
    ├── Section-3-image.png
    ├── agent-image.png
    └── career.png
```

## Sections

- **Header** — Fixed navigation bar with logo, nav links, and sign in / download buttons
- **Hero** — Headline and intro screenshot with a download CTA
- **Logo Garden** — Trusted company logos (Stripe, OpenAI, Linear, Datadog, Nvidia, Figma, Ramp, Adobe)
- **Features** — Three feature sections: Agent, Autocomplete (Tab), and a third feature panel
- **Reviews** — Six-card testimonial grid from notable developers
- **Stay on the Frontier** — Three-card grid covering Models, Semantic Search, and Enterprise
- **Changelog** — Recent release cards
- **Team / Careers** — Team description with a careers CTA
- **Recent Highlights** — Blog post feed
- **CTA** — Final download call-to-action
- **Footer** — Links, social icons, theme switcher, and language selector

## How to Run

Open `index.html` directly in a browser — no build step or server required.

```bash
xdg-open index.html   # Linux
open index.html       # macOS
```

## Tech Used

- HTML5
- CSS3 (Flexbox, CSS Grid)
- Font Awesome 4.7 (via CDN) for icons
- SVG icons (inline)
